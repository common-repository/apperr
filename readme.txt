== Description ==

Apperr is very flexible way to get you started with Android and iOS Apps for your WordPress website.
We support WordPress Blog app and WooCommerce based e-commerce App. With Apperr we are providing your very unique set of features

==Apperr Features==

**App Templates**
App templates are like themes. With different App templates you can create different kind of App as per your need.
Each template comes with unique set of features. You can utilize these feature across the App screens.

**Drag and Drop App screen builder**
We provide drag and drop component based builder right into the WordPress dashboard. 
With Screen Builder you can add content to your App screens. Drag and Drop builder provides different components. Each component comes with unique set of features.
You can use those components do design you App screen. With different App templates you will get wide range of components to choose.

**Control from Dashboard**
Plugin provides all the Settings in central place. You can control core settings of the App right from you WordPress dashboard.

**App Build on the Cloud**
Once you are ready you can submit app for build. Our App builder is hosted in cloud. It will build your App in few minutes. You will be able to download App from your WordPress Dashboard.

== Installation ==

Use built in automatic installer or,

1. Copy the plugin folder into /wp-content/plugins
2. Activate plugin via the plugins page in the WordPress admin
3. Go to Settings to configure.
